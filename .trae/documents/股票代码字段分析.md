# 股票代码字段分析

## 数据流程

### 1. 数据库 daily 表
```sql
SELECT DISTINCT code FROM daily
-- 数据库中 code 字段是 6 位数字字符串: '000001', '600000', '300001'
```

### 2. get_all_stock_codes_cached() 函数
```python
# 原始：从数据库获取
cursor.execute('SELECT DISTINCT code FROM daily ORDER BY code')
codes = [row[0] for row in cursor.fetchall()]
# 返回: ['000001', '000002', '600000', ...] (6位数字)

# 修改后：添加前缀
for row in cursor.fetchall():
    code = row[0]
    if code.startswith('6'):
        codes.append('sh' + code)  # sh600000
    elif code.startswith(('000', '001', '002', '003', '300')):
        codes.append('sz' + code)   # sz000001
# 返回: ['sz000001', 'sz000002', 'sh600000', ...]
```

### 3. scan_single_stock() 函数
```python
def scan_single_stock(code, days, db, analyzer):
    # code 可能是 'sz000001' 或 'sh600000'

    # 去前缀，查询数据库
    db_code = code.replace('sh', '').replace('sz', '')  # 000001

    # 查询K线数据
    kline_data = db.get_kline_data(db_code, days=days)

    # 缠论分析
    result = analyzer.analyze(kline_data, db_code)

    # 构建信号字典 - 使用 db_code
    signals.append({
        '股票代码': db_code,  # 这里应该是 000001 (6位)
        ...
    })

    return signals
```
