# 背驰比较逻辑优化计划

## 一、逻辑说明

以下降趋势ABCD为例，最低点分别是abcd：

### 比较步骤

**步骤1：B和A比较**
- 如果AB形成中枢（差距<1%）→ 用C和A比较
- 如果AB不形成中枢（差距>1%）→ 比较A和B的背驰

**步骤2：C和A比较**
- 如果AC形成中枢（差距<1%）→ 用D和A比较
- 如果AC不形成中枢（差距>1%）→ 比较A和C的背驰

**步骤3：D和A比较**
- 比较A和D的背驰

    downs = [t for t in trends if t['type'] == '下降']

    for i, t in enumerate(downs):
        if i == 0:

        compare_idx = i - 1

        while compare_idx >= 0:
