## 二、代码复用

现有代码中已有中枢判断逻辑（在 `detect_divergence` 函数）：
```python
pl = abs(curr['price_low'] - prev['price_low']) / prev['price_low'] * 100
in_z = pl < 2  # 如果差距小于2%，认为在中枢内
```

**复用此逻辑**，但用于控制是否比较，而不是仅仅标记。

## 三、比较逻辑

### 规则
1. 比较下降趋势时，先检查低点差距
2. 如果差距不超过2%（`pl < 2`），跳过前一个趋势，继续向前找
3. 如果差距超过2%，则进行比较
4. 如果前面没有可比较的了，则必须比较
