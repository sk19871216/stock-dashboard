# 股票数据看板

基于 Streamlit 的自用股票情报追踪与预测系统。

## 功能特性

- 📊 **数据更新**: 自动更新自选股日K数据、涨跌停、板块资金、龙虎榜
- 🔍 **情报追踪**: 自选股管理、触发指标筛选、大盘分析、涨跌停分析
- 🤖 **股票预测**: 基于技术指标的明日股价预测
- 📝 **复盘分析**: 预测与实际对比，准确率统计

## 技术栈

- **前端**: Streamlit
- **数据获取**: mootdx, akshare
- **数据存储**: SQLite
- **技术分析**: pandas, numpy

## 安装

1. 确保已安装 Python 3.8+

2. 安装依赖:

```bash
pip install -r requirements.txt
```

主要依赖:
- streamlit>=1.28.0
- mootdx>=0.8.0 (需要安装，通达信数据源)
- akshare>=1.12.0 (备选数据源)
- pandas>=2.0.0
- plotly>=5.18.0

## 运行

```bash
cd f:\trae_project\股票看板
streamlit run app.py
```

或直接:

```bash
streamlit run app.py
```

应用将在浏览器中打开 (默认: http://localhost:8501)

## 使用说明

### 1. 数据更新

首次使用需要更新数据:
1. 进入"数据更新"页面
2. 添加自选股（如需要）
3. 点击"更新所有自选股数据"

### 2. 情报追踪

- 在"自选股"标签页管理股票
- 使用"触发指标"筛选符合条件的股票
- 查看"大盘分析"了解市场趋势
- 查看"涨跌停"了解热点板块

### 3. 股票预测

1. 选择或输入股票代码
2. 查看技术指标分析
3. 获取明日预测价格和方向
4. 保存预测记录

### 4. 复盘分析

1. 每天收盘后更新实际价格
2. 查看预测与实际的对比
3. 记录复盘分析
4. 统计准确率，改进预测方法

## 技术指标说明

系统使用以下技术指标进行预测:

- **MA**: 移动平均线 (5日、20日、60日)
- **MACD**: 指数平滑异同移动平均线
- **KDJ**: 随机指标
- **BOLL**: 布林带
- **RSI**: 相对强弱指标

## 常见问题

### Q: mootdx 安装失败?
A: 可以只使用 akshare 作为数据源，修改 `utils/data_fetcher.py` 中的数据获取逻辑

### Q: 数据更新失败?
A: 检查网络连接，可能是数据源服务器问题，稍后重试

### Q: 如何添加新的股票?
A: 在情报追踪页面的"自选股"标签页添加
